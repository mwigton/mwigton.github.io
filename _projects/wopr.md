A fun retro project recreating the iconic War Games computer interface. Implemented on authentic vintage hardware (IBM PC XT with DOS 2.0). Used OpenWatcom compiler and vim for development, then deployed via USB floppy drive adapter. The computer is on display at the Cold War Museum. Full source available on GitHub.
